# 基于 ELF2 的多传感融合 AI 智能座舱安全交互系统

本项目是一款基于 ELF2 开发板（RK3588 平台）开发的多传感融合 AI 智能座舱安全交互系统。系统集成了多种传感器数据，利用 AI 算法对座舱环境及驾驶员状态进行智能感知与安全交互。
